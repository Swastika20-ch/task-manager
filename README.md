# Smart Task Manager

A modern and responsive Task Management Web Application built using HTML, CSS, and JavaScript. The application helps users organize daily activities efficiently by allowing them to create, track, complete, and delete tasks while storing data locally in the browser.

## Project Overview

Managing tasks effectively is essential for productivity. This project provides a simple yet powerful solution for organizing daily activities through an intuitive user interface and persistent storage using the browser's Local Storage API.

The application enables users to maintain a personal task list, track progress, and manage completed tasks without requiring any backend services or external databases.

## Features

* Add new tasks instantly
* Mark tasks as completed
* Delete unnecessary tasks
* Real-time task statistics
* Persistent task storage using Local Storage
* Responsive design for desktop and mobile devices
* Clean and user-friendly interface
* Dynamic task updates without page reloads

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API



### Run the Application

Open `index.html` in any modern web browser.

No additional dependencies or installations are required.

## Functionalities

### Add Task

Users can add new tasks to their task list.

### Complete Task

Tasks can be marked as completed to track progress.

### Delete Task

Tasks can be removed whenever they are no longer needed.

### Local Storage Integration

All tasks are automatically saved in the browser and remain available even after refreshing or reopening the application.
